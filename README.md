# Estetica Joyce Arraiss

## About

Static mobile-first landing page for an aesthetics clinic. The project uses HTML, CSS and Vue 3 via CDN to keep the first version simple, portable and easy to deploy on static hosting providers.

## Features

- Mobile-first layout starting at 320px
- Sticky header with animated hamburger menu
- Service sections for facial, body and beauty treatments
- Touch-friendly gallery with lightbox and swipe gestures
- Floating WhatsApp CTA and direct call action
- Lightweight PWA assets with manifest and service worker
- SEO and Open Graph base metadata

## Installation

1. Clone or copy the project files.
2. Serve the folder with a static server.

Example:

```powershell
npx serve .
```

## Usage

1. Update the clinic data in `js/app.js`.
2. Replace placeholder images in `img/placeholder/`.
3. Open `index.html` through a local server for service worker support.
